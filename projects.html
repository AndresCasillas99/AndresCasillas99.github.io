Some of my favorite projects:
  • German Learning App [2025](Python, Streamlit)
    I built a Streamlit app to learn German, featuring vocabulary, grammar, and exercises. It includes a vocabulary trainer and a grammar section with explanations and examples.
  • LovaszThetaNumber [2025](Julia, Jupyter)
    The Lovász Theta number of an undirected graph can be found by solving an eigenvalue problem. An interesting property of it is given by Lovász's Sandwich Theorem, which links the clique number and the chromatic number of a graph and its compliment graph. For perfect graphs, it is enough to find the Lovász Theta number with accuracy of less than one half to know the chromatic number and maximum independent size of a graph. This allows for efficient polynomial time algorithms for finding maximum independent sets and optimal colorings. It also behaves well with the graph strong product, and gives an upper bound for the Shannon Capacity of a graph. In a nutshell, it's worth cumputing *wink emoji* (found it ---> 😉).
  • Conway's Game of Life (Processing) [2024](Processing)
    Just for fun, I recreated Conway’s famous cellular automaton in Processing to watch simple rules create surprisingly complex patterns. Processing is great for cellular automata visualizations.
  • Langton's Ant (Processing) [2024](Processing)
    I did this little simulation of Langton’s Ant to see how a tiny rule can create order and chaos from an empty grid. Several interesting open questions (which I have not diven into yet) arise from this simple automaton.
  • Rule30 [2024](Julia, Jupyter)
    An efficient algorithm for computing the n-th row of Wolfram's Rule 30. It uses an indexing approach as the one in CSR sparse matrix storage.
  • Rule30-Pseudo-Random-Number-Generator [2024](Julia, Jupyter)
    A (pseudo)random number generator using the central column values of Wolfram's Rule 30. I once heard Wolfram uses the same method in the pseudo-random number generator of Mathematica (is this true?).
  • PoissonProblemGalerkinMethod [2024](Python, Jupyter)
    Approximates the solution of the weak form of the Poisson problem in the unit square with zero boundary values using the Galerkin method. Includes plots to visualize approximate solutions. This is kind of a dummy project of the real deal, which I coded Fortran 💀.
  • BuffonsNeedle
    For fun, I coded Buffon’s Needle experiment to estimate 𝜋. It was one of the first programs in which I got to play around with structs and somewhat more "serious" programming.
